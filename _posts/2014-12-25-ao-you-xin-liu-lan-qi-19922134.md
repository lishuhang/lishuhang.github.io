傲游 17 号在美国加州发布了一个号称是“目前市面上最快的 Windows 浏览器”。傲游表示， [名为 Nitro 的新版浏览器](https://link.zhihu.com/?target=http%3A//usa.maxthon.com/nitro/) 启动速度比以往快 3 倍，而且加载和抓取网页的速度要比任何已经存在的 Windows 浏览器都要快。傲游 CEO 陈明杰表示：“我们用户当中的 80%都认为，速度是他们对于浏览器的第一追求，而 Nitro 会很好的满足这个需求。”

那么 Nitro 到底是怎样满足速度需求的呢？根据一些媒体的下载评测，它采用了最新的 Chrome 35 版内核，同时界面基本上不可定制，没有加载项，甚至设置面板都不完整。除了最基本的地址栏，标签页，后退前进这些功能之外，别的什么都没有。

这不禁让人想起 IE9 刚刚发布时的样子。跟 IE8 相比，基本上把所有可见到的元素都砍掉了，只在地址栏的右侧留下了三个非常不起眼的小图标，而且地址栏和标签栏放到了同一列当中。我们当时都开玩笑说，IE 算是真正的认清了自己只是用来下载别的浏览器的浏览器的，反正大家也不把我设为默认，那么我就尽量简单一点，大家用我下完了别的浏览器，我就可以退休了。

说真的，傲游 Nitro 目前也跟 IE9 以后的版本具有同样的特点，就是基本上不支持任何加载项，没有任何界面元素，没有可以定制的功能，设置面板聊胜于无。除了一个空荡荡的框架之外，别的什么都没有。

而为什么 IE 和现在的傲游都要这么改？这是因为谷歌浏览器的推出。谷歌在推出自己浏览器的时候，选择的名字 Chrome，在英文当中的意思就是“框架”。谷歌当时就希望人们浏览互联网的时候，能是一种沉浸式的体验，不应该因浏览器本身的架构而喧宾夺主。那是什么时候？那个时候，PC 版浏览器都还停留在傲游 2 和 Firefox 那样臃肿不堪，各种功能都有的基础上。

这也是傲游必须以新品牌推广新的浏览器的原因：这个名字和 Firefox 一样，已经牢牢地和吃内存，占资源，假死这样的负面印象联系到了一起。就算改叫“傲游极速版”也不行。

在 Chrome 以后，简洁就成为一代又一代浏览器更新换代的核心诉求。但浏览器真正简洁是不可能的，否则就有点自寻死路了。Chrome 后来也照样加入插件，而变得更加臃肿。所以 IE 就接过了“简洁”的大旗。而 IE 发展几代以后也放回了加载项，傲游在他们两个都变的有点臃肿的时候再次“简洁”了一遍。

可以想见，以后傲游 Nitro 还是得把加载项，设置面板，换肤等功能陆续加进来的。那么谁来接替 Nitro 的位置，继续“简洁”下去呢？
